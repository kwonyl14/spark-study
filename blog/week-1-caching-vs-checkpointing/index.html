1мЈјм°Ё: Spark Cachingкіј Checkpointingмқҳ м°ЁмқҙмҷҖ мӢӨм „ нҷңмҡ©лІ•

February 22, 2026 В· 4 min read
Kwon Younglin
Data Engineer

мҠӨнҢҢнҒ¬(Spark) м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ к°ңл°ңн•ҳлӢӨ ліҙл©ҙ JoinмқҙлӮҳ м •л ¬(Sort) к°ҷмқҖ л¬ҙкұ°мҡҙ(Expensive) м—°мӮ° кІ°кіјлҘј м—¬лҹ¬ лІҲ мһ¬мӮ¬мҡ©н•ҙм•ј н•ҳлҠ” мғҒнҷ©мқҙ л°ҳл“ңмӢң л°ңмғқн•ңлӢӨ. мқҙл•Ң м•„л¬ҙ мІҳлҰ¬лҸ„ н•ҳм§Җ м•Ҡмңјл©ҙ м•Ўм…ҳ(Action)мқҙ нҳём¶ңлҗ  л•Ңл§ҲлӢӨ лҚ°мқҙн„°лҘј мІҳмқҢл¶Җн„° лӢӨмӢң мқҪкі  кі„мӮ°н•ҳлҠ” м—„мІӯлӮң л№„нҡЁмңЁмқҙ л°ңмғқн•ңлӢӨ.

мқҙ л¬ём ңлҘј н•ҙкІ°н•ҳлҠ” л‘җ к°Җм§Җ н•өмӢ¬ л¬ҙкё°к°Җ л°”лЎң мәҗмӢұ(Caching) кіј мІҙнҒ¬нҸ¬мқёнҢ…(Checkpointing) мқҙлӢӨ. л‘җ кё°лІ•мқҖ л№„мҠ·н•ҙ ліҙмқҙм§Җл§Ң лӮҙл¶Җ лҸҷмһ‘ мӣҗлҰ¬мҷҖ лӘ©м Ғмқҙ мҷ„м „нһҲ лӢӨлҘҙлӢӨ. мҪ”л“ңлҘј нҶөн•ҙ мқҙ л‘ҳмқҳ м°ЁмқҙлҘј нҷ•мқён•ңлӢӨ.

1. Caching: л№ лҘё мһ¬мӮ¬мҡ©мқ„ мң„н•ң л©”лӘЁлҰ¬ мөңм Ғнҷ”вҖӢ

мәҗмӢұмқҖ кі„мӮ°мқҙ лҒқлӮң DataFrameмқҙлӮҳ RDDлҘј л©”лӘЁлҰ¬лӮҳ л””мҠӨнҒ¬м—җ мһ„мӢңлЎң м ҖмһҘн•ҙ л‘җлҠ” кё°лІ•мқҙлӢӨ. к°ҖмһҘ нҒ° нҠ№м§•мқҖ лҚ°мқҙн„°мқҳ кі„мӮ° мЎұліҙ(Lineage)лҘј к·ёлҢҖлЎң мң м§Җн•ңлӢӨлҠ” м җмқҙлӢӨ. мҰү, мәҗмӢңлҗң лҚ°мқҙн„°к°Җ лӮ м•„к°Җл©ҙ мҠӨнҢҢнҒ¬лҠ” м–ём ңл“  мІҳмқҢл¶Җн„° лӢӨмӢң кі„мӮ°н•ҙм„ң ліөкө¬н•  мҲҳ мһҲлӢӨ.

1-1. DataFrame мәҗмӢұкіј StorageLevelвҖӢ

мҠӨнҢҢнҒ¬лҠ” мғҒнҷ©м—җ л§һкІҢ лҚ°мқҙн„°лҘј м–ҙл””м—җ, м–ҙл–»кІҢ м ҖмһҘн• м§Җ кІ°м •н•ҳлҠ” лӢӨм–‘н•ң StorageLevelмқ„ м ңкіөн•ңлӢӨ.

import org.apache.spark.sql.SparkSession
import org.apache.spark.storage.StorageLevel

val spark = SparkSession.builder()
  .appName("CachingDemo")
  .config("spark.memory.offHeap.enabled", "true")
  .config("spark.memory.offHeap.size", 10000000)
  .master("local")
  .getOrCreate()

val flightsDF = spark.read.option("inferSchema", "true").json("src/main/resources/data/flights")
val orderedFlightsDF = flightsDF.orderBy("dist") // л¬ҙкұ°мҡҙ м—°мӮ° к°Җм •

// лӢӨм–‘н•ң мәҗмӢұ мҳөм…ҳ м Ғмҡ©
orderedFlightsDF.persist(
  // 1. л©”лӘЁлҰ¬м—җ мӣҗліё к°қмІҙлЎң м ҖмһҘ (к°ҖмһҘ л№ лҰ„, OOM мң„н—ҳ)
  // StorageLevel.MEMORY_ONLY 
  
  // 2. л””мҠӨнҒ¬м—җл§Ң м ҖмһҘ (л©”лӘЁлҰ¬ м Ҳм•Ҫ, I/OлЎң мқён•ҙ лҠҗлҰј)
  // StorageLevel.DISK_ONLY 
  
  // 3. л©”лӘЁлҰ¬м—җ лЁјм Җ мұ„мҡ°кі , л„ҳм№ҳл©ҙ л””мҠӨнҒ¬лЎң Spill (DF кё°ліёк°’)
  // StorageLevel.MEMORY_AND_DISK 

  // 4. к°Җл№„м§Җ м»¬л үн„°(GC)лҘј нҡҢн”јн•ҳлҠ” Off-Heap л©”лӘЁлҰ¬ м ҖмһҘ (Tungsten м—”м§„ нҷңмҡ©)
  StorageLevel.OFF_HEAP 
)

orderedFlightsDF.count() // мІ« лІҲм§ё м•Ўм…ҳ: кі„мӮ° нӣ„ мәҗмӢң л©”лӘЁлҰ¬м—җ м Ғмһ¬лҗЁ
orderedFlightsDF.count() // л‘җ лІҲм§ё м•Ўм…ҳ: мәҗмӢңм—җм„ң л°”лЎң мқҪм–ҙмҷҖм„ң нӣЁм”¬ л№ лҰ„!

// мәҗмӢң л©”лӘЁлҰ¬ н•ҙм ң
orderedFlightsDF.unpersist()

1-2. RDD м§Ғл ¬нҷ” мәҗмӢұ (л©”лӘЁлҰ¬ лӢӨмқҙм–ҙнҠё)вҖӢ

мҳӨлһҳлҗң RDD APIлҘј мӮ¬мҡ©н•  л•ҢлҠ” _SER(Serialized) мҳөм…ҳмқҙ н•„мҲҳм ҒмқҙлӢӨ.

val flightsRDD = orderedFlightsDF.rdd

// мһҗл°” к°қмІҙ лҢҖмӢ  л°”мқҙнҠё л°°м—ҙлЎң м§Ғл ¬нҷ”н•ҳм—¬ л©”лӘЁлҰ¬ мҡ©лҹү к·№м Ғ м Ҳк°җ
flightsRDD.persist(StorageLevel.MEMORY_ONLY_SER)
flightsRDD.count()

RDDлҠ” л¬ҙкұ°мҡҙ JVM к°қмІҙлҘј мғқм„ұн•ҳкё° л•Ңл¬ём—җ MEMORY_ONLYлҘј м“°л©ҙ OOMмқҙ н„°м§Җкё° мүҪлӢӨ. н•ҳм§Җл§Ң DataFrameмқ„ м“°л©ҙ мҠӨнҢҢнҒ¬ м№ҙнғҲлҰ¬мҠӨнҠё(Catalyst) м—”м§„мқҙ м•Ңм•„м„ң нҡЁмңЁм Ғмқё л°”мқҙл„ҲлҰ¬ нҸ¬л§·мңјлЎң м ҖмһҘн•ҙмЈјкё° л•Ңл¬ём—җ көімқҙ RDDлЎң ліҖнҷҳн•ҙм„ң мәҗмӢұн•  н•„мҡ”лҠ” м—ҶлӢӨ.

2. Checkpointing: DAGлҘј лҒҠм–ҙлӮҙлҠ” лӢЁл‘җлҢҖвҖӢ

мІҙнҒ¬нҸ¬мқёнҢ…мқҖ л©”лӘЁлҰ¬ л¶ҖмЎұмқҙлӮҳ л…ёл“ң мһҘм• лҘј лҢҖл№„н•ҙ, лҚ°мқҙн„°лҘј мһ„мӢңк°Җ м•„лӢҢ HDFSлӮҳ лЎңм»¬ л””мҠӨнҒ¬ к°ҷмқҖ мҳҒкө¬м Ғмқҙкі  м•Ҳм „н•ң м ҖмһҘмҶҢм—җ л¬јлҰ¬м ҒмңјлЎң кё°лЎқн•ҙ лІ„лҰ¬лҠ” кё°лІ•мқҙлӢӨ.

к°ҖмһҘ мӨ‘мҡ”н•ң м°ЁмқҙлҠ” кі„мӮ° мЎұліҙ(Lineage, DAG)лҘј мӢ№л‘‘ лҒҠм–ҙлІ„лҰ°лӢӨлҠ” кІғмқҙлӢӨ.

2-1. Checkpointing мҪ”л“ң кө¬нҳ„вҖӢ

мІҙнҒ¬нҸ¬мқёнҢ…мқ„ мӮ¬мҡ©н•ҳл Өл©ҙ л°ҳл“ңмӢң мҠӨнҢҢнҒ¬ м»Ён…ҚмҠӨнҠё(sc)м—җ м ҖмһҘн•  л””л үнҶ лҰ¬лҘј лЁјм Җ м§Җм •н•ҙм•ј н•ңлӢӨ.

val sc = spark.sparkContext
// мІҙнҒ¬нҸ¬мқёнҠё м ҖмһҘ кІҪлЎң м„Өм • (мӢӨл¬ҙм—җм„ңлҠ” HDFSлӮҳ S3 кІҪлЎң мӮ¬мҡ©)
sc.setCheckpointDir("spark-warehouse")

val orderedFlights = flightsDF.orderBy("dist")

// checkpoint() мһҗмІҙк°Җ м•Ўм…ҳ(Action)мқ„ нҠёлҰ¬кұ°н•ҳл©° л””мҠӨнҒ¬м—җ лҚ°мқҙн„°лҘј м“ҙлӢӨ.
val checkpointedFlights = orderedFlights.checkpoint() 

checkpointedFlights.count() // л””мҠӨнҒ¬м—җм„ң л°”лЎң мқҪм–ҙмҳҙ

2-2. мӢӨн–ү кі„нҡҚ(Physical Plan) л№„көҗ л¶„м„қвҖӢ

мәҗмӢұкіј мІҙнҒ¬нҸ¬мқёнҢ…мқҳ к·јліём Ғмқё м°ЁмқҙлҠ” explain()мқ„ нҶөн•ҙ мӢӨн–ү кі„нҡҚмқ„ м—ҙм–ҙлҙӨмқ„ л•Ң лӘ…нҷ•н•ҳкІҢ л“ңлҹ¬лӮңлӢӨ.

[мқјл°ҳ DF лҳҗлҠ” мәҗмӢұлҗң DFмқҳ мӢӨн–ү кі„нҡҚ]

orderedFlights.explain()
/*
== Physical Plan ==
*(1) Sort [dist#16 ASC NULLS FIRST], true, 0
+- *(1) Project [_id#7, arrdelay#8, ... ]
   +- BatchScan[...] JsonScan Location: InMemoryFileIndex[...]
*/

мң„мІҳлҹј мәҗмӢңлҘј м“°лҚ”лқјлҸ„ мҠӨнҢҢнҒ¬лҠ” "м—¬м°Ён•ҳл©ҙ лӮҙк°Җ нҢҢмқјл¶Җн„° лӢӨмӢң мқҪм–ҙм„ң Sortн• кІҢ"лқјл©° м „мІҙ кі„нҡҚ(Lineage)мқ„ кёёкІҢ л“Өкі  мһҲлӢӨ.

[мІҙнҒ¬нҸ¬мқёнҠёлҗң DFмқҳ мӢӨн–ү кі„нҡҚ]

checkpointedFlights.explain()
/*
== Physical Plan ==
*(1) Scan ExistingRDD[_id#7,arrdelay#8, ... ]
*/

мІҙнҒ¬нҸ¬мқёнҠё мқҙнӣ„мқҳ лҚ°мқҙн„°н”„л Ҳмһ„мқҖ мқҙм „мқҳ Sort, Project, JsonScan к°ҷмқҖ ліөмһЎн•ң кіјкұ°лҘј мҷ„м „нһҲ мһҠм–ҙлІ„лҰ°лӢӨ. мҳӨм§Ғ л””мҠӨнҒ¬м—җ м ҖмһҘлҗң кІ°кіјл¬ј(ExistingRDD)мқ„ мғҲлЎңмҡҙ мӢңмһ‘м җ(Source)мңјлЎң мӮјлҠ”лӢӨ.

3. мҡ”м•Ҫ л°Ҹ мӢӨм „ кІ°лЎ : м–ём ң л¬ҙм—Үмқ„ мҚЁм•ј н• к№Ң?вҖӢ

кө¬л¶„Caching (мәҗмӢұ)Checkpointing (мІҙнҒ¬нҸ¬мқёнҢ…)
м ҖмһҘ мң„м№ҳл©”лӘЁлҰ¬ мӨ‘мӢ¬ (м„Өм •м—җ л”°лқј л””мҠӨнҒ¬ нҳјмҡ©)мҳҒкө¬ м ҖмһҘмҶҢ (HDFS, S3, лЎңм»¬ л””мҠӨнҒ¬ л“ұ)
Lineage (мЎұліҙ)мң м§ҖлҗЁ (мһҘм•  мӢң мІҳмқҢл¶Җн„° лӢӨмӢң кі„мӮ°)лӢЁм ҲлҗЁ (кіјкұ° м—°мӮ° кё°лЎқ мӮӯм ң)
мҶҚлҸ„л§Өмҡ° л№ лҰ„ (нҠ№нһҲ л©”лӘЁлҰ¬ Hit мӢң)мғҒлҢҖм ҒмңјлЎң лҠҗлҰј (л””мҠӨнҒ¬ I/O л°ңмғқ)
лӘ©м Ғл°ҳліө м—°мӮ° мҶҚлҸ„ н–ҘмғҒмһҘм•  ліөкө¬, кёҙ м—°мӮ° мІҙмқё лҒҠкё°

рҹҡҖ Cachingмқ„ мҚЁм•ј н•  л•Ң (Performance Focus)вҖӢ

  • лҸҷмқјн•ң мһЎ(Job) м•Ҳм—җм„ң лЁёмӢ лҹ¬лӢқ лӘЁлҚёмқ„ л°ҳліө н•ҷмҠөмӢңнӮ¬ л•Ң.
  • лЈ©м—… н…Ңмқҙлё”(Lookup Table)мІҳлҹј мһҗмЈј мЎ°мқёлҗҳлҠ” мһ‘мқҖ DataFrameмқ„ мһ¬мӮ¬мҡ©н•  л•Ң.
  • м—°мӮ° мҶҚлҸ„лҘј мҘҗм–ҙм§ңм•ј н•ҳлҠ” мғҒнҷ©мқј л•Ң.

рҹӣЎпёҸ Checkpointingмқ„ мҚЁм•ј н•  л•Ң (Reliability Focus)вҖӢ

  • л°°м№ҳ мһ‘м—… мӢңк°„мқҙ л„Ҳл¬ҙ кёём–ҙм„ң мӨ‘к°„м—җ м—җлҹ¬к°Җ лӮ¬мқ„ л•Ң мІҳмқҢл¶Җн„° лӢӨмӢң лҸҢлҰ¬лҠ” кІғмқҙ мһ¬м•ҷмқё кІҪмҡ°.
  • DataFrameмқҳ withColumnмқҙлӮҳ join ліҖнҷҳмқҙ мҲҳмӢӯ, мҲҳл°ұ лІҲ кІ№міҗм„ң мӢӨн–ү кі„нҡҚ(DAG)мқҙ л„Ҳл¬ҙ кёём–ҙм§Җкі  л©”лӘЁлҰ¬лҘј л§Һмқҙ лЁ№м–ҙ л“ңлқјмқҙлІ„м—җм„ң StackOverflowErrorк°Җ н„°м§Ҳ мң„н—ҳмқҙ мһҲмқ„ л•Ң. (мқҙл•Ң мІҙнҒ¬нҸ¬мқёнҠёлҘј н•ң лІҲм”© м°Қм–ҙмЈјл©ҙ DAGк°Җ мҙҲкё°нҷ”лҗҳм–ҙ м—җлҹ¬лҘј л§үмқ„ мҲҳ мһҲлӢӨ.)
  • Spark Structured Streaming мҡҙмҳҒ мӢң мһҘм•  ліөкө¬лҘј мң„н•ҙ.

рҹ’Ў мӢӨл¬ҙ мҡҙмҳҒ нҢҒвҖӢ

нҳ„м—…м—җм„ңлҠ” кёҙ лҚ°мқҙн„° нҢҢмқҙн”„лқјмқёмқ„ м„Өкі„н•  л•Ң, мӨ‘к°„мӨ‘к°„ л¬ҙкұ°мҡҙ м…”н”Ң(Shuffle)мқҙ лҒқлӮҳлҠ” м§Җм җм—җ мәҗмӢұкіј мІҙнҒ¬нҸ¬мқёнҢ…мқ„ н•Ёк»ҳ мӮ¬мҡ©н•ҳкё°лҸ„ н•ңлӢӨ. лҚ°мқҙн„°лҘј л©”лӘЁлҰ¬м—җ мҳ¬л Ө л№ лҘҙкІҢ мһ¬мӮ¬мҡ©н•ҳл©ҙм„ңлҸ„, л””мҠӨнҒ¬м—җ м•Ҳм „н•ҳкІҢ л°ұм—…н•ҙл‘җм–ҙ DAGлҘј мһҗлҘҙкі  мһҘм• м—җ лҢҖл№„н•ҳлҠ” нӣҢлҘӯн•ң м „лһөмқҙлӢӨ.
